**Project Name:** Bezier Spline Project

**Description:** This project is a Unity-based implementation of a Bezier spline, a mathematical curve used to model smooth curves in 2D and 3D space.

<img width="1020" height="699" alt="b" src="https://github.com/user-attachments/assets/fc7e25ab-d2ae-4bef-b960-699215b808a7" />

**Features:**

* Implementation of a Bezier spline using C# and Unity
* Adjustable curve points using the Unity editor
* Optional looping of the curve

**Requirements:**

* Unity 2021 or later
* C# programming language

**Getting Started:**

1. Clone or download the project repository.
2. Open the project in Unity.
3. Navigate to the `Assets/Scripts` folder and open the `BezierSpline.cs` script.
4. Adjust the curve points and looping settings to your liking.

**Usage:**

* Attach the `BezierSpline` script to a GameObject in your Unity scene.
* Use the `points` array to adjust the curve points.
* Use the `loop` variable to enable or disable looping of the curve.
